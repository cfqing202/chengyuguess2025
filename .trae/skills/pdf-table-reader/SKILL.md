# PDF 表格读取器

用于读取 PDF 文件并提取其中所有表格内容的技能。

## 使用场景

- 用户需要分析 PDF 文档中的表格
- 用户上传了 PDF 文件需要提取数据
- 用户问"你能读取 PDF 吗"相关问题

## 使用方法

1. **提供 PDF 文件路径** - 用户需提供 PDF 文件的绝对路径
2. **读取并解析** - 使用 pdf.js 或其他 PDF 解析工具读取文件
3. **提取表格** - 识别并提取文档中的所有表格
4. **输出结果** - 以结构化格式返回表格数据

## 输出格式

默认输出 Markdown 表格格式，也可根据需要输出 JSON 格式。

## 注意事项

- 目前仅支持读取 PDF 文件，不支持 .docx 格式
- 如果是 .docx 文件，建议用户转换为 PDF 或复制文字内容
